# Urban Issue Detection using YOLOv8

This project detects urban issues such as damage and trash using a YOLOv8 object detection model.

## Features
- Data preprocessing and annotation conversion
- Data augmentation
- YOLOv8 model training
- Cross-validation
- Performance evaluation

## Technologies
- Python
- OpenCV
- NumPy
- Pandas
- PyTorch
- Ultralytics YOLOv8
- Matplotlib

## Results
- mAP@0.5: 0.64
- Precision: 0.57
- Recall: 0.65
